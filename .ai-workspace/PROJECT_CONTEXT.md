# Project Context

## Project Name
SynapseIDE Urban Analytics

## Default Repository
mustafaras/SynapseIDE_urban_analytics

## Project Purpose
This repository is the main development workspace for SynapseIDE Urban Analytics. The project focuses on an AI-supported, professional, VS Code-like urban analytics interface with a refined technical UI/UX direction.

## Core Principles
- Preserve existing functionality.
- Do not make unnecessary structural changes.
- Keep changes modular, readable, and reversible.
- Prefer TypeScript-safe and maintainable code.
- Avoid unnecessary dependencies.
- Keep UI professional, technical, and restrained.
- Maintain consistency across layout, theme, panels, menus, and interaction states.

## UI/UX Direction
- VS Code-like workspace logic.
- Professional urban analytics dashboard/interface.
- Refined glassmorphism only where useful.
- Avoid flashy, excessive visual effects.
- Prioritize clarity, usability, hierarchy, and responsive behavior.

## AI Working Rule
Any AI assistant working on this repository must first review the files under `.ai-workspace/`, especially:

- PROJECT_CONTEXT.md
- ARCHITECTURE.md
- DECISIONS.md
- WORKFLOW.md
- ROADMAP.md

## Git Workflow
- Do not commit directly to main/master unless explicitly instructed.
- Create a feature branch for every task.
- Prepare a pull request for review.
- Include changed files, risks, and test notes in every PR summary.
